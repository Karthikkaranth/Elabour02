                                   E-LABOUR: Digital Workforce Hiring Application

Overview
E-LABOUR is a web-based labour hiring application designed to bridge the gap between contractors and workers through a user-friendly digital platform. The system enables contractors to post job opportunities and allows workers to search, view, and apply for available jobs efficiently. 

Features
•	Contractor registration and login 
•	Worker registration and login 
•	Job posting with role, salary, and location details 
•	Job search and application system for workers 
•	Efficient data management 
•	User-friendly interface 
•	Improved communication between contractors and workers 


Technologies Used
•	Python Flask 
•	HTML 
•	CSS 
•	JavaScript 
•	MongoDB Database 


Project Objective
The primary objective of E-LABOUR is to simplify the traditional labour hiring process by providing a digital platform that makes workforce hiring faster, more organized, and accessible for both contractors and workers. 
How It Works
1.	Contractors create accounts and post job opportunities. 
2.	Workers register on the platform and browse available jobs. 
3.	Workers apply for suitable job openings. 
4.	The platform facilitates communication and workforce management in a streamlined manner.
